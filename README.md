## Quick start

For an in depth guide of the installation and running, go to [`DEVELOPMENT.md`](./DEVELOPMENT.md).

If you have everything nicely installed, then you can quickly do:

### Frontend

```bash
cd frontend
yarn install
yarn dev 
```

### Backend

```bash
cd backend
poetry install
poetry run dev
```
